Respositório de Curso Cypress
Ministrado pelo professor Francisco Aquino na Udemy

* Cypress

### Instalação
Clonando o projeto
Url: https://github.com/Mizael2019/curso-cypress.git

### Instalando as dependências:
* npm install cypress ou npm i cypress


### Rodando os testes

- Rodando testes diretamente no browser:<br>
$ npm run cypress:open

- Rodando os testes em segundo plano (headless):<br>
$ npm run cypress:run
